# CytoClaw Analysis Report Generator

## 触发条件

分析流程完成后（Harmony 矫正 + 出图之后），**自动生成** HTML 报告。无需用户请求。

## HTML 模板结构

生成报告时，写一个 Python 脚本创建 `Analysis_Report.html`：

```python
import base64, datetime

def img_to_b64(path):
    with open(path, 'rb') as f:
        return base64.b64encode(f.read()).decode()

WS = os.path.expanduser('~/.openclaw/workspace')

# 收集所有生成的图
figures = {}
import os
for name in ['out_leiden_umap.png', 'out_batch_umap.png',
             'out_harmony_umap.png', 'out_nature_umap.png',
             'out_heatmap.png', 'out_cd8a.png']:
    path = f'{WS}/{name}'
    if os.path.exists(path):
        figures[name] = img_to_b64(path)
```

## 报告正文必须包含的章节

按此顺序，每个章节用 `.card` 包裹：

### 1. 概览统计（stat-grid）

```html
<div class="stat-grid">
  <div class="stat-box">
    <div class="value">8,117</div>
    <div class="label">质控后细胞数</div>
  </div>
  <div class="stat-box">
    <div class="value">7</div>
    <div class="label">Leiden 聚类数</div>
  </div>
  <!-- silhouette, 中心距, Harmony 迭代次数 等 -->
</div>
```

### 2. 质控摘要

- 原始细胞数 → 过滤后
- MT 阈值、基因数范围
- 用 `.tag-green` / `.tag-red` 标注通过/异常

### 3. 批次效应检测

- Before Harmony UMAP（嵌入 base64 图片）
- 定量指标（silhouette, 中心距）
- 判定结论

### 4. Harmony 矫正结果

- After Harmony UMAP（嵌入 base64 图片）
- 前后指标对比表（用 `<table>`）

### 5. 聚类与 Marker 分析

- 莫兰迪色系 UMAP
- 差异表达热图
- Top marker 基因表

### 6. 特征表达图

- CD8A 等关键 marker 的特征表达 UMAP

### 7. Footer

```html
<div class="footer">
  Generated by CytoClaw · {datetime} · scanpy {version} · harmonypy {version}
</div>
```

## 图片嵌入方式

所有图片必须以 base64 嵌入，不依赖外部文件：

```html
<img src="data:image/png;base64,{b64_string}" alt="描述文字">
```

## 输出路径

```
~/.openclaw/workspace/Analysis_Report.html
```

生成后通过飞书文件 API 发送给用户。

## 关键约束

- 单文件自包含，在手机浏览器上可读
- 不使用任何 CDN 或外部资源
- 配色与 TOOLS.md 中的莫兰迪方案保持一致
- 中文为主，专业术语首次出现加英文注释
- 报告长度控制在 1 屏~3 屏，不堆砌参数细节
